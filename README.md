
# cryptocurrency APP
Complete Flutter Application with Riverpod & Freezed + Dio for API REST.



<img src="screenshots/cover.png" >

## Features
- API REST (CryptoWatch)
- Linear Graph View (Hour, Day, Week, etc)
- OHLC Graph
- Search
- Light / Dark Theme
- Multi Language
